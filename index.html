엑셀처럼 입력하기

셀을 클릭해서 직접 입력하거나, 진짜 엑셀에서 범위를 복사(Ctrl+C)한 뒤 첫 칸을 클릭하고 붙여넣기(Ctrl+V)하면 여러 행·열이 한 번에 채워집니다. 힌트랜딩 URL 칸에는 원하는 URL을 자유롭게 입력하세요.
'일괄등록 원본양식' 다운로드는 기존에 쓰시던 파일과 같은 컬럼 구성(#, 광고주ID, 캠페인타입, 캠페인명, 시작일시, 종료일시, 전체참여인원, 일최대참여인원, 광고단가, 광고예산, 힌트랜딩URL, 플레이스/쇼핑URL, 키워드, 문제, 정답 {NSN4는 이미지 포함})으로 만들어집니다. 문제·정답·캠페인명은 자동 계산된 값이 그대로 채워집니다. 단, 이 파일은 원본 데이터(플레이스명·명소·상품정보 등)가 들어있지 않아 다시 업로드해서 계속 편집하는 용도로는 쓸 수 없습니다. 계속 편집하시려면 "원본데이터 백업 다운로드"를 이용해주세요.
팁: 셀 하나를 클릭하면 오른쪽 아래에 작은 사각형(채우기 핸들)이 나타납니다. 그 핸들을 아래로 드래그하면 엑셀처럼 같은 값이 여러 행에 자동으로 채워집니다.

파트너 정산 연동 (구글 스프레드시트)

로그인한 파트너 키를 기준으로 자동 연동됩니다. 파트너별 플레이스단가 / 쇼핑단가가 구글 스프레드시트 기준으로 적용되어 "전체참여한도 × 단가"가 계산됩니다.
기록 시점은 ① QuizClick API 등록 성공 시와 ② '일괄등록' 엑셀 다운로드 시 두 곳이며, 기록 전에 항상 확인 팝업으로 정산 금액을 보여드리고 "예"를 눌러야 실제로 기록됩니다.

QuizClick 관리자 화면

구글 스프레드시트("소진원본"/"파트너단가"/"충전내역") 데이터를 기준으로, 파트너별 예상비용/소진/충전/잔액 현황을 확인합니다.

파트너별 충전 / 소진금액 / 잔액

소진수량·금액은 "소진원본" 시트(광고주/파트너사/날짜/캠페인타입/소진수량)와 "파트너단가" 시트(유형별 단가)를 기준으로 자동 계산됩니다. 충전금액은 아래에서 직접 입력하거나, "충전내역" 시트에 바로 추가하셔도 됩니다.
